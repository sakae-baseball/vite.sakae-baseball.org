# 協会概要と役員一覧

市民野球協会は、市民スポーツの振興と青少年の健全育成を目的として活動しています。本大会の運営をはじめ、審判養成講座やジュニアクリニックなど地域密着の取り組みを行っています。

## 協会のミッション

1. 市民が安心して野球を楽しめる環境づくり
2. 次世代選手の育成と地域チームの支援
3. 公正な試合運営とルール啓蒙活動

## 組織体制

- 会員数: 320名 (2025年4月現在)
- 登録チーム: 24チーム
- 主な活動拠点: 市営第一・第二球場、野球協会トレーニングセンター
