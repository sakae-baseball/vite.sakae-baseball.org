# 「野球ねっと」へのチーム登録のお願い

日頃は、栄区野球協会の運営に協力いただき、ありがとうございます。

栄区民野球大会では2024年度より、参加チームおよび競技者の「野球ねっと」への登録をお願いしております。

「野球ねっと」は、全日本軟式野球連盟が加盟している日本野球協議会の加盟団体が共同で運営しています。

本日(2026/3/17)、未登録および登録手続きが完了していないチーム代表者のメールアドレス宛に、以下の内容で野球ねっとのチーム登録に関するメールを送信しております。

```
From: noreply@yakyu-net.jp
Subject: 認証メール
```

メールを受け取られたチームの代表の方は、以下のリンク先の手順を参照の上、「野球ねっと」へのチームの登録の完了をお願いします。

- [https://static.sakae-baseball.org/resource/yakyunet-2026.pdf](https://static.sakae-baseball.org/resource/yakyunet-2026.pdf) (PDF)

なおすでにチーム登録ずみの場合は、行き違いですのでご容赦ください。
